# Contributing to MoringaPulse

First off, thank you for considering contributing to MoringaPulse! It's people like you that make it possible to promote sustainability through technology.

## 🌈 Our Mission

To provide a professional, data-driven platform that highlights the nutritional and environmental benefits of *Moringa oleifera*.

## 🚀 How Can I Contribute?

### Reporting Bugs
If you find a bug, please open an issue with a clear description and steps to reproduce.

### Suggesting Enhancements
We love new ideas! Whether it's a new visualization or a better calculation model, let us know.

### Pull Requests
1. Fork the repo.
2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'feat: Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## 📜 Coding Standards

- Use **Conventional Commits**.
- Maintain **100% TypeScript** type safety.
- Write unit tests for all new calculation logic.
- Follow the existing Tailwind CSS design patterns.
